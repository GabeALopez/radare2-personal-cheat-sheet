# Personal Cheet Sheet

## Debugging

### Break, Start, run, and then run again

1. db main
2. dc
3. ood
4. ood (To start again)

## Some Usage

### Renaming functions

1. pdf ( or pdr if in debug mode) @ [address of function]
2. afn [function_name]

### Renaming variables

1. pdf - show function
2. afvn [var name in asm] [personal var name]

### List all functions

- afl
